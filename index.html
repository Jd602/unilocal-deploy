Políticas y condiciones UniLocal
Las siguientes son las políticas para el uso de nuestros servicios:

  • El usuario final para interactuar con los negocios o hacer comentarios debe estar registrado.
  • El usuario final para interactuar con los negocios o hacer comentarios debe estar registrado.
  • Para realizar el registro el usuario debe contar con los datos requeridos (cédula, nombre, fecha de nacimiento, entre otros).
  • Por cada correo solo se puede registrar un usuario.
  • Para registrar un negocio el usuario ya deberá contar con una cuenta registrada.
  • Para el registro es obligatorio suministrar toda la información requerida en el formulario.
  • Todo usuario deberá tener respeto para el uso de la plataforma, el no cumplimiento de esto genera sanciones como suspensión temporal o la eliminación de la cuenta, esto ya será de manejo del moderador que maneje el caso.
  • Para resolver dudas puede contactar al correo este_no_es_un_correo@gmail.com
